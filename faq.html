What is a non exception language?+

A non exception language the creative language has to be the geo associated language. For example:

(NON exception) - Geo: Spain language settings: Spanish

What is an exception language?+

An exception language is one that would allow the creatives in the campaign to be in either English or the associated Geo language when the language settings are on non English. For example:

Exception language - Geo: Germany language settings: German

Ads serving to: German profiles creatives CAN be either: German OR English
